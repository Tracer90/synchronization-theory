10. Техническая спецификация ядра

Guard (observe → authorize → apply)

ПравилоСмысл
No effect without authorizationКаждое действие проходит проверку перед выполнением
No write outside scopeЗапись только в разрешённые пути. .env, .git, backups, MEMORY.md — заблокированы
No success without verificationУспех не засчитывается без подтверждения
Heuristic is not realityЭвристика — не истина. Числа 3,4,5 — не абсолют, а текущая компрессия
LLM suggests; code guard decidesLLM предлагает, код принимает решения. Защита от галлюцинаций

Метрики

МетрикаФормулаСмысл
Sync ScoreSSₙ = 1 − |predicted − actual| + SSₙ₋₁Рекурсивная метрика. Sync Score системы включает Sync Score её подсистем. Всё остальное — производные
SyncErrorSEₙ = |predicted − actual| + SEₙ₋₁Рекурсивная ошибка. Чем глубже, тем точнее модель
Invariant StrengthΣ(domain_count × avg_weight)Сила инварианта. Порог: >2.5

Режимы запуска

python live.py              # probe, exit (safe default)
python live.py --once --llm  # single cycle with LLM
python live.py --loop --llm  # autonomous loop
python live.py --write ...   # allow write operations
